## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- Node
- Web browser

### Setup
- Open terminal porgram
- Clone this repositiory by running `git clone https://github.com/kimothokamau/mathmagicians.git `


### Install
- Run `npm install`


### Usage
- Run `npm start`
